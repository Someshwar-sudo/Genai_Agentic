# Generative AI & Agentic AI Learning Laboratory

Welcome to my personal repository dedicated to mastering Generative AI and Agentic AI workflows. This repository serves as a centralized hub for my practice files, hands-on tutorials, code snippets, and reference architectures as I upgrade my skills in advanced AI technologies.

## Author's Note
**Every file, script, notebook, and line of code in this repository was written entirely by myself.** This codebase represents my hands-on practice, custom implementations, and active experimentation as I build my skills from scratch in GenAI and Agentic AI systems.

## Project Overview
The transition from prompt engineering to autonomous AI agents represents the next frontier in technology. This repository documents my personal learning journey, tracking my progress from foundational Large Language Model (LLM) implementations to complex, multi-agent systems that can plan, use tools, and collaborate to solve tasks.

## Repository Structure
*   `01-foundations/` - My basic LLM API integrations, prompt engineering techniques, and tokenization scripts.
*   `02-rag-systems/` - My custom Retrieval-Augmented Generation code using vector databases.
*   `03-agentic-frameworks/` - My hands-on experiments with LangChain, LangGraph, CrewAI, and Autogen.
*   `04-tools-and-function-calling/` - Scripts I wrote to teach LLMs to use external APIs, web scrapers, and databases.
*   `05-projects/` - Mini-projects built by me combining GenAI and autonomous agents.
*   `reference-guides/` - My personal cheatsheets, research paper summaries, and architecture diagrams.

##  Key Learning Objectives
*   **LLM Fine-Tuning & Prompting:** Mastering advanced prompting strategies (Chain-of-Thought, ReAct).
*   **RAG Architecture:** Building efficient context-aware applications with semantic search.
*   **Agentic Workflows:** Designing single and multi-agent systems with tool-use capabilities.
*   **Memory & State Management:** Implementing short-term, long-term, and episodic memory in agents.
*   **Evaluation & Monitoring:** Testing AI outputs for hallucinations and optimizing response latency.

##  Tech Stack & Tools
*   **Languages:** Python
*   **LLM Providers:** OpenAI (GPT), Anthropic (Claude), Ollama (Local Models)
*   **Frameworks:** LangChain, CrewAI, LangGraph
*   **Vector DBs:** Chroma, FAISS
*   **Deployment:** Streamlit, Chainlit (for quick UI prototyping)
